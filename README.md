# CS Upgrade Simulator

Одиночная локальная браузерная игра на React + TypeScript + Zustand.

- В игре есть только внутриигровые монеты.
- Монеты нельзя купить за реальные деньги.
- Скины нельзя вывести, обменять или подключить к Steam.
- Все предметы используются только как элементы локальной коллекции.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

Данные скинов генерируются в `src/data/skins/*.json`, изображения - в `public/skins/*.svg`.
Цены указаны как ориентировочные внутриигровые значения в монетах, стилизованные под гривневую оценку известных предметов.
